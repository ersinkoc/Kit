API Reference

Complete API documentation for all @oxog/kit modules

@oxog/kit/utils

String, array, object, date, math, and random utilities

  • string - String manipulation functions
  • array - Array operations
  • object - Object utilities
  • date - Date formatting and manipulation
  • random - Random generation

@oxog/kit/validation

Schema validation, sanitization, and transformation

  • validate - Schema validation
  • sanitize - Input sanitization
  • transform - Data transformation

@oxog/kit/security

JWT, hashing, encryption, and crypto utilities

  • jwt - JSON Web Tokens
  • hash - Password hashing
  • crypto - Encryption utilities

@oxog/kit/core

Configuration, logging, events, context, errors, hooks

  • log - Structured logging
  • createEventEmitter - Type-safe events
  • createConfig - Configuration management
  • createContext - Context propagation

@oxog/kit/async

Queue, retry, circuit breaker, rate limiter, mutex

  • createQueue - Task queue
  • retry - Retry with backoff
  • createCircuitBreaker - Circuit breaker
  • createRateLimiter - Rate limiting
  • createMutex - Mutex/semaphore

@oxog/kit/observability

Metrics, health checks, distributed tracing

  • metrics - Counters, gauges, histograms
  • createHealthChecker - Health checks
  • createTracer - Distributed tracing

@oxog/kit/network

HTTP client, WebSocket, SSE clients

  • http - HTTP client
  • createWebSocket - WebSocket client
  • createSSE - SSE client

@oxog/kit/data

Cache, store, session management

  • createCache - In-memory cache
  • createStore - Key-value store
  • createSession - Session management

@oxog/kit/parsing

JSON, URL, query string, MIME type parsing

  • json - Safe JSON parsing
  • url - URL utilities
  • querystring - Query string handling
  • mime - MIME type detection
